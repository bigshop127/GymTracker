# Phase 5 — 進度圖表

> 前置：Phase 4 已有歷史資料；`lib/e1rm.ts`、`lib/volume.ts` 可用。

## 目標
針對單一動作呈現訓練進度趨勢，並顯示個人紀錄（PR）。

## 規格
1. 進度頁（`/progress`）：
   - 上方選一個動作（複用 Phase 2 的動作選擇）。
   - 用 Recharts 畫趨勢線，X 軸＝時間（每次有練到該動作的訓練），可切換三種指標：
     - **預估 1RM（E1RM）**：取該次該動作所有有效組的最大 E1RM（用設定的公式）。
     - **單次最大重量**：該次該動作有效組的最大 weight。
     - **單次總容量**：該次該動作有效組的 Σ(weight×reps)。
   - 顯示單位依設定（kg/lb 換算只在顯示層）。
2. PR 區塊：顯示該動作歷史最佳 E1RM（與發生日期），可加最大重量 PR。
3. 資料不足（該動作只練過 0–1 次）時給友善提示，不要讓圖崩掉。

## 驗收標準
- 選「槓鈴臥推」：隨多次紀錄、重量上升，E1RM 線往上。
- 切換三種指標圖都正確、暖身組未被計入。
- PR 數字＝歷史最高 E1RM、日期正確。
- 切換 kg/lb 後圖與數字同步換算。

## 注意
- 計算跨多次訓練的聚合，先把資料整理成 `{date, value}[]` 再餵圖，邏輯放 `lib/` 方便測試。
